# The system for drawing prizes

**Курсовая работа**

CLI-система для случайного выбора победителя из списка участников.
Реализует «колесо фортуны» с анимацией в консоли, сохраняет историю розыгрышей в JSON-файл и предоставляет простое текстовое меню для взаимодействия с пользователем.

## О проекте

Консольное приложение для розыгрыша призов.
Позволяет загружать список участников из JSON-файла, запускать анимированное «колесо фортуны», определять случайного победителя и сохранять историю всех розыгрышей.

## 🚀 Установка и запуск

### Предварительные требования

- **Node.js** версии 18 или выше

Проверить версию или установлен ли Node.js:

```bash
node -v
```

Если не установлен Node.js →

### Установка Node.js (для Visual Studio Code)

1. Скачайте и установите Node.js с официального сайта:  
   [https://nodejs.org/](https://nodejs.org/) (рекомендуется LTS-версия 18+)

2. Откройте **Visual Studio Code** и запустите встроенный терминал:  
   `Terminal` → `New Terminal` (или нажмите `` Ctrl + ` ``)

3. Проверьте версию Node.js:
   ```bash
   node -v
   ```

## Запуск

1.  **Клонируйте репозиторий:**
    ```bash
    git clone https://github.com/lemoneden/the-drawing-system.git
    ```
2.  **Перейдите в папку проекта:**
    ```bash
    cd the-drawing-system
    ```
3.  **Откройте терминал в вашем редакторе кода и запустите программу:**
    ```bash
    node main.js
    ```

## 🏗️ Структура проекта

```
├── data/
│   ├── participants.json   – список участников (загружается пользователем)
│   └── history.json        – история розыгрышей (создаётся автоматически)
├── modules/
│   ├── lottery.js          – функция случайного выбора победителя
│   ├── history.js          – чтение/запись истории, показ истории
│   └── interface.js        – меню, анимация, цветной вывод
└── main.js                 – точка входа, запуск приложения
```
